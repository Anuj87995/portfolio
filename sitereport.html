Site Report

This website was created as part of my learning journey in web development. Throughout this project, I learned the fundamentals of HTML, CSS, and JavaScript, and how they come together to build a fully functional website.

Development Experience

The process started with designing the layout and structure of the website. I chose a clean and minimalistic approach, using a simple navigation bar and a well-structured gallery. Learning how to style elements using CSS was a key aspect of this project.

Design Choices

The font style was selected for readability, and I used a color scheme that ensures clarity and contrast. I was inspired by websites such as Apple and Dribbble for their sleek designs and user-friendly interfaces.
